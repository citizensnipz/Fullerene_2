# Prompt: Retro

Use after meaningful work to improve system quality and harness quality.

## Instructions to agent
1. Review:
   - `ai/logs/SESSION_LOG.md`
   - `ai/logs/CHANGELOG_AI.md`
   - relevant decisions/issues
2. Summarize what worked and what failed.
3. Propose 1-3 harness improvements.
4. Convert improvements into concrete actions.

## Guardrails
- Base on evidence, not preference.
- Keep changes small and actionable.
- Avoid process bloat.

## Output format
- Context
- Wins
- Pain points
- Evidence
- Action items (owner + due date)
- Harness files to update
