# Glossary

Shared vocabulary for humans and AI agents.

## Terms
- **Fullerene**
  - Experimental local-first cognitive architecture for LLM agents.
- **AI Harness**
  - Structured `/ai` memory + workflow layer used by multiple tools.
- **Local-first**
  - Prioritizes local operation/state as primary source of control.
- **VERIFIED**
  - Confirmed directly from repo/docs/runtime evidence.
- **UNVERIFIED**
  - Not yet confirmed; must not be treated as fact.
